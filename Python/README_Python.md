# Python Implementation (Object-Oriented)

## Description

This implementation calculates the mean, median, and mode of a list of
integers. It follows an object-oriented programming approach using
Python.

## What It Contains

-   `StatisticsCalculator` class
-   Encapsulated methods: `mean()`, `median()`, and `mode()`
-   Use of Python lists and dictionaries
-   Built-in sorting
-   Frequency counting for mode
-   Proper handling of invalid or empty input

## How to Run

Ensure Python 3 is installed.

``` bash
python3 stats.py
```

Enter integers separated by spaces when prompted.
